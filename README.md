# 项目概述 java437

本项目是一个基于Springboot和Vue的音乐推荐管理系统，通过协同过滤算法为用户提供个性化的音乐推荐服务。系统主要分为用户和管理员两个角色，用户可享受丰富的音乐播放与管理功能，管理员则负责维护系统数据和音乐内容。

## 技术栈

### 后端

- Springboot
- MyBatis

### 前端

- Vue
- ElementUI
- ECharts

## 功能模块

### 用户功能

- 首页推荐歌单
- 歌单分类分页浏览
- 歌手分类分页浏览
- 查看和收藏我的音乐
- 音乐搜索（支持按歌手、歌曲、歌单名搜索）
- 头像与用户信息修改
- 歌曲播放，进度条调整
- 歌词显示
- 收藏歌曲
- 歌曲下载

### 管理员功能

- 系统首页统计数据展示
- 用户管理
- 歌手管理
- 歌曲管理（包括音源、歌词、评论的修改）
- 音乐上传

## 系统角色

- 用户：使用前台界面进行音乐播放和个性化操作。
- 管理员：通过后台界面进行系统管理和音乐内容维护。

## 运行环境

- JAVA版本：JDK 1.8
- 开发工具：IDEA
- 数据库类型：MySQL 8
